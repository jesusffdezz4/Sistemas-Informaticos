Software

Es la parte lógica del sistema informático. Dentro del software podemos distinguir:

  • Software base, encargado de hacer funcionar el hardware, e incluye el sistema operativo, controladores y el firmware.

  • Software de aplicación, constituido por el resto de aplicaciones informáticas que permite convertir al ordenador en una herramienta útil para el usuario.
