Software privativo

Es un software que no es libre ni de código abierto. Dentro de este software hay modos de distribución muy utilizados:

  • Freeware: Es simplemente un software que se distribuye de forma gratuita.
  • Shareware: Son versiones gratuitas de software que se ofrecen para la evaluación, por lo que tienen algunas restricciones (días de uso, tiempo continuo de uso, etc.). La característica fundamental de este software es que no pretende ser totalmente funcional, sino solo permitir la evaluación del producto.
  • Crippleware: Son versiones limitadas de software, normalmente gratuitas, que ofrecen una funcionalidad reducida en comparación con la versión completa. Normalmente se denominan versiones Lite. La principal diferencia con el shareware es que estas versiones están destinadas a ser funcionales, aunque no ofrezcan todas las funciones.

Por otro lado, dependiendo de quién sea el destinatario de la licencia, se distinguen las siguientes:

  • El EULA, que es una licencia para un único usuario.
  • OEM, que es una licencia asociada al software incluido en un ordenador. Por ejemplo, si compramos un ordenador que tiene instalado un sistema Windows, normalmente incorpora una licencia OEM.
  • Licencias corporativas o de volumen, que son licencias que se conceden a usuarios de la misma empresa u organismo. Un ejemplo de esta licencia es Azure for Education, que Microsoft ofrece a instituciones académicas que enseñan informática.
